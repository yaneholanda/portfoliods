Analisando a Pesquisa de Satisfação de clientes de Telecomunicações, com Python

April 19, 2021

Este artigo tem como objetivo analisar os dados disponíveis em dados.gov.br, de pesquisas de satisfação e qualidade recebida, realizadas com os clientes de operadoras de telecomunicações. Esta pesquisa foi realizada pela ANATEL. Como estudante de Engenharia de Telecomunicações, decidi utilizar Data Science para analisar os dados da minha área de atuação, a partir dos dados retirados desta página.

Vamos iniciar preparando o ambiente, com as bibliotecas adequadas a este trabalho e carregando os dados para o Notebook.

Em seguida, analisaremos os primeiros dados do arquivo.

Agora, vamos iniciar as nossas primeiras análises.

Esta pesquisa realizada pela Anatel levou em consideração alguns serviços distintos. Vamos descobrir quantos e quais são eles?

Foram levados em consideração os Serviços de Banda Larga Fixa, Telefonia Fixa, Telefonia Móvel Pós-Paga, Telefonia Móvel Pré-paga e TV por Assinatura. Qual será a média de notas dadas pelos usuários para estes serviços?

Esta é a média geral de todos os serviços. Perceba que a pesquisa levou em consideração alguns indicadores, como você pode ter percebido quando analisamos inicialmente os dados. Para fins de consulta individual, vamos escolher um indicador e fazer a análise dele.

Qual Operadora possui os melhores Canais de Atendimento, de acordo com os usuários?

É a Operadora “Nossa TV”, no estado do Rio Grande do Sul.

Até aqui, obtivemos uma visão geral dos nossos dados. Neste momento, partiremos para as visualizações gráficas.

Utilizaremos a biblioteca Matplotlib.

Como estamos utilizando o indicador Canais de Atendimento, vamos visualizar a média de cada um por operadora.

Comparando com as notas recebidas em todos os indicadores, podemos perceber uma certa proximidade entre as duas análises.

Vamos descobrir as notas dos serviços avaliados em Canais de Atendimento?

Agora, vamos escolher uma operadora aleatoriamente e realizar uma análise entre os anos de 2015–2020 de seu indicador Canais de Atendimento.

Podemos perceber um aumento nas notas atribuídas aos Canais de Atendimento da Oi ao longo dos anos, pelos usuários dos seus serviços. Realizando uma análise mais geral, vamos analisar todas as operadoras ao longo dos anos de acordo com seus serviços.

Percebemos um aumento nas notas do serviço de Banda Larga Fixa. Em contrapartida, todos os outros serviços tiveram suas notas diminuídas entre os anos de 2018 e 2019. Esperamos que este cenário tenha mudado para os anos posteriores. Caro leitor, espero que tenha gostado deste artigo.
